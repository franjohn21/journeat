### Journeat - Find places to eat on your journey

> Built by Francis John and Andy Chiu during the DBC Hackathon 12/14/2014

#### Technologies:
- Sinatra Backend
- Vanilla Javascript/AngularJS
- jQuery/Ajax
- Consumes Google Maps API
- Consumes Yelp API
